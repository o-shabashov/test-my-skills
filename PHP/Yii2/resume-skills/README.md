## Installation
* Clone repository

```bash
git clone https://github.com/o-shabashov/test-my-skills.git
cd PHP/Yii2/resume-skills
```

* Install composer dependencies:

```bash
composer install
```

* Create MySQL database `mydb` and run migrate

```bash
./yii migrate
```

* Run server:
```bash
./yii serve
```

1. [Yii2](https://github.com/yiisoft/yii2)
